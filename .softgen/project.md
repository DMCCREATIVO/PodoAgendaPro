# PODOS PRO – Sistema Podológico Completo

## Vision
Sistema clínico unitario premium para clínicas podológicas o podólogos independientes. Un ecosistema completo que integra agenda inteligente, ficha podológica profesional, portal de pacientes y panel administrativo — todo con diseño de nivel SaaS empresarial. Destinado a podólogos que buscan software real, usable en consulta diaria y presentable comercialmente.

## Design
**Palette**: `--primary: 217 91% 60% (blue)`, `--background: 210 40% 98% (light)`, `--foreground: 222 47% 11% (dark slate)`, `--muted: 220 13% 91% (gray)`, `--accent: 142 71% 45% (green)`, `--destructive: 0 84% 60% (red)`

**Fonts**: Heading: Plus Jakarta Sans (weights 600, 700), Body: Inter (weights 400, 500, 600)

**Style**: Premium SaaS meets healthcare — soft neumorphism, glassmorphism on sidebars/modals, generous whitespace, smooth micro-interactions, rounded corners (16-24px), subtle shadows, tinted neutrals

## Features
- **Landing público**: Hero, servicios, podólogos, beneficios, testimonios, CTA
- **Reserva inteligente**: Stepper moderno (servicio → podólogo → fecha → datos → confirmación)
- **Disponibilidad compartible**: Página marketing por podólogo con agenda visual
- **Panel Admin**: Dashboard KPI, gestión citas/podólogos/pacientes/cobros, configuración
- **Panel Podólogo**: Mi Día, Atención Clínica (ficha podológica completa), Mis Pacientes
- **Portal Paciente**: Mis citas, historial permitido, pagos
- **Auth visual**: Selector elegante (Admin/Podólogo/Paciente)
